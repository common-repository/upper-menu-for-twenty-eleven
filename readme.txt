Moves the default Twenty Eleven menu to the top of the page with a nice CSS.

== Description ==

This plugin moves the default Twenty Eleven menu via CSS to the top of the page.
It adds rounded corners and respects the header image. 

The plugin is intended for only-image headers, without text, to show them a bit prettier. 

== Installation ==

Opt 1:

1. Go to Plugins -> Menu -> Add new
2. Search for `upper menu`
3. Install the plugin
4. Activate the plugin

Opt 2:

1. Download the plugin from http://downloads.wordpress.org/plugin/upper-menu.zip
2. Upload `upper-menu` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
